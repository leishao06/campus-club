# 社团招新智能匹配平台（可交互原型）

依据仓库内 `prd.md` 实现的前端演示：三端入口、极速测评、规则过滤 + 标签相似度匹配、社团主页六模块、一键投递与进度、对比与收藏、社团/校方工作台。

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开终端提示的本地地址（一般为 `http://localhost:5173`）。

## 构建

```bash
npm run build
npm run preview
```

## 使用说明

1. 首页选择 **新生端 / 社团管理端 / 校方监管端**。
2. 新生：完成 **极速测评** → **匹配结果** → **发现/详情** → **我的档案** 后 **一键投递**（最多 5 个）。
3. 社团端：在 **报名与录取** 中推进报名状态，新生在 **我的报名** 中可见节点变化。
4. 校方端：**数据大屏**、**投诉工单**。
5. 数据保存在浏览器 `localStorage`，首页可 **清除本地演示数据**。

## 技术栈

- Vite 5 + React 18 + TypeScript
- React Router 6
- 无后端，匹配逻辑在 `src/lib/matching.ts`
